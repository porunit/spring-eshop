# spring-eshop
Spring-Eshop repository is a study full-stack e-commerce application built using:
### Back-End
<ul>
<li>Spring</li>
<li>Spring Security </li> 
<li>Spring Boot</li>
<li>Spring MVC</li>
<li>Hibernate</li>
<li>PostgreSQL</li>
<li>Flyway for data migration</li>
</ul>

### Front-End
<ul>

<li>CSS</li>

<li>HTML</li>
</ul>
